# TripletNet

## Dataset
Download dataset VOC2007 and VOC2012, put them under `VOCdevkit` directory:
```
VOCdevkit
-| VOC2007
   -| Annotations
   -| ImageSets
   -| JPEGImages
   -| SegmentationClass
   -| SegmentationObject
-| VOC2012
   -| Annotations
   -| ImageSets
   -| JPEGImages
   -| SegmentationClass
   -| SegmentationObject
```


## Usage
### train:
`python train.py`
